# sales-and-revenue-project
Sales & Revenue Analysis
    Project Overview:
This project analyzes an eCommerce dataset to understand sales performance and revenue trends. The goal is to clean the data, perform analysis, and build dashboards that provide meaningful business insights.
Tools Used:
Python (Pandas, NumPy, Jupyter Notebook)
MySQL
Power BI
VS Code
GitHub
Project Steps:
Data Cleaning (Python) – Removed missing values, duplicates, and prepared the dataset.
Data Analysis (SQL) – Wrote queries to analyze revenue, sales, and product performance.
Data Visualization (Power BI) – Built interactive dashboards to present insights.
Skills Demonstrated:
Data Cleaning • SQL Analysis • Data Visualization • Dashboard Building • Business Insights
